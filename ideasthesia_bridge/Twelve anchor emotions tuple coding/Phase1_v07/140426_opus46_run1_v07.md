WORD: FEAR
RAW: A sudden vertical plane rising directly in front of you, very close. It has no width you can walk around; it simply occupies the forward direction. Behind it, space compresses into a thin wedge. The plane does not move, but you are being pushed toward it.
CODE: F:convergence N:contrastive/b# D:stable P:den2D
NOTE: Form could be "collapse" but the plane holds; it doesn't fold inward. It's a wall, not an implosion. Stable because the attractor state is frozen confrontation, not disintegration.

WORD: ANXIETY
RAW: A loose orbit that never closes. The path curves back toward its origin but misses it slightly each time, tracing a spiral that neither tightens nor expands. The centre of the orbit is empty; the moving point keeps searching for it.
CODE: F:loop N:isolated/b~ D:recursive P:med1D
NOTE: "Drifting" was tempting for D, but the recursion is the defining feature; it revisits near-same territory compulsively. Neighbourhood is isolated because the loop is self-referential; its surroundings are irrelevant to its behaviour.

WORD: JOY
RAW: An outward expansion from a single origin, roughly spherical but uneven, like a starburst with unequal arms. The expansion is fast and decelerating. Space around the origin stretches open.
CODE: F:point N:gradient/b∅ D:stable P:sp0D
NOTE: Point because everything radiates from one locus. Gradient because the boundary dissolves outward; there is no edge. Stable because the attractor is the expanded state, not the expansion event.

WORD: EXCITEMENT
RAW: A vector with a wobble in it. It has strong forward direction but oscillates laterally as it travels, like a arrow vibrating along its shaft mid-flight. The space ahead is open and pulling.
CODE: F:vector N:gradient/b~ D:drifting P:med1D
NOTE: Borderline between "drifting" and "recursive" for D. Chose drifting because the oscillation doesn't return to previous states; it's forward-biased instability. The wobble prevents "stable."

WORD: GRIEF
RAW: A massive dense sphere sitting in an otherwise empty field. It does not move. Space around it bends inward slightly, as though the sphere has weight that warps its surroundings. You orbit it; it doesn't orbit you.
CODE: F:convergence N:isolated/b# D:stable P:den3D
NOTE: "Collapse" is close, but grief's attractor state is the mass, not the collapsing process. The thing has already fallen inward and now simply sits there. Sharp boundary because grief is distinctly itself; nothing blends into it easily.

WORD: SADNESS
RAW: A shallow, wide basin. A gentle concavity in a flat plane, curving downward so slowly you don't notice you're descending until you're at the lowest point. The edges of the basin are indistinct.
CODE: F:convergence N:gradient/b~ D:drifting P:med2D
NOTE: Sadness vs grief: grief is a dense object, sadness is a landscape feature. Drifting because the attractor is a slow settling with no fixed final depth. Gradient because sadness bleeds into adjacent states without hard transitions.

WORD: CONTENT
RAW: A flat plane, very slightly convex, like the top of a gentle hill. No movement. The plane extends evenly in all directions. You are at the highest point but the curvature is so mild it barely registers.
CODE: F:point N:clustered/b~ D:stable P:sp0D
NOTE: Point because there is a single locus (you, here). Clustered because contentment is defined by comfortable proximity to its neighbours; it sits among related states without tension. The mildness of everything is the signal.

WORD: PEACEFUL
RAW: An absence of geometry. A volume of space where vectors have been removed. What remains is uniform extension with no gradients, no centres, no edges. It is less a shape than the condition of a space from which shapes have been cleared.
CODE: F:point N:gradient/b∅ D:stable P:sp0D
NOTE: Difficult to code because the phenomenology is anti-formal. Point by default (simplest form for a non-structure). Gradient because it dissolves into everything. The key difference from content: content is a mild presence, peaceful is a structured absence.

WORD: DESPERATION
RAW: Multiple vectors firing outward from a single point in rapid succession, each in a different direction, each terminating abruptly in a wall. The space is a closed chamber. The vectors bounce and die; new ones fire.
CODE: F:vector N:contrastive/b# D:recursive P:den2D
NOTE: "Collapse" was considered for F, but the geometry is vectors-hitting-walls, not inward folding. Recursive because the firing-and-failing cycle repeats. Contrastive with sharp boundary because desperation is defined entirely against the constraints that produce it.

WORD: TERRIFIED
RAW: A single point that has become infinitely dense. All spatial extension has been pulled into it. There is no surrounding space; the point has consumed its own neighbourhood. It does not move because there is nowhere to move to.
CODE: F:collapse N:isolated/b# D:terminal P:sp0D
NOTE: This is the only word so far I'd code as terminal. The attractor state is a singularity; there is no next phase from within the state itself. Collapse rather than convergence because dimensionality has been lost, not just gathered.

WORD: PANICKED
RAW: A fast, tight, irregular oscillation with no axis. A point jerking in three dimensions without pattern, covering a small volume at high speed. The motion is entirely local; it goes nowhere. The surrounding space is irrelevant because the point cannot perceive it.
CODE: F:loop N:isolated/b# D:recursive P:den3D
NOTE: Loop rather than vector because there is no net displacement. The motion is frantic but trapped. Recursive because it repeats without resolving. Panicked vs terrified: terrified is a collapse to stillness, panicked is a collapse to noise. Different geometries of the same neighbourhood.
