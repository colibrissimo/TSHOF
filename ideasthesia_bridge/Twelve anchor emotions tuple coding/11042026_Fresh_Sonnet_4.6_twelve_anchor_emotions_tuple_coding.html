Pair 1 — neg · high arousal
Fear
F: vector N: contrastive⁺b D: drifting P: sp1D
A directed threat-response: single-point, moving fast toward an exit. Hard boundary — self vs. danger. Arousal is high but the geometry is simple. Drifting because fear shifts with the threat object.
Anxiety
F: loop N: isolated⁻b D: recursive P: med1D
No clear threat-object; the representational content circles. Isolated because anxiety is internally generated — other people don't feed into it. Recursive because it returns to the same anticipatory position.
Pair 2 — neg · high arousal
Terrified
F: convergence N: clustered⁺b D: stable P: den2D
Extreme fear with a fixed, overwhelming object — all activation converges on that locus. Dense and stable because the threat is so vivid it holds steady. Harder to perturb than fear precisely because the object is so dominant.
Panicked
F: collapse N: contrastive⁻b D: drifting P: med2D
Panic is executive collapse — control structure breaks down. F:collapse not because the emotion ends but because the representational field fails to organise. Drifting: responses are incoherent, shifting. Boundary gone (⁻b) because the self-schema is overwhelmed.
Pair 3 — pos · high arousal
Joy
F: loop N: clustered⁻b D: recursive P: den2D
Joy is a sustained positive loop — it circles back to itself, returns to the good state. Clustered because joy is relational (shared experience). No hard boundary (⁻b): joy expands outward. Dense cluster, but stable-recursive not drifting.
Excitement
F: vector N: gradient⁻b D: drifting P: med2D
Excitement is anticipatory — forward-directed toward a goal-object. Vector form: moving toward something. Drifting because excitement tracks the anticipated event, shifting as the situation evolves. Gradient neighbourhood: blends into adjacent positive states easily.
Pair 4 — pos · low arousal
Content
F: point N: isolated⁻b D: stable P: sp0D
Contentment is a resting state — no aspiration, no movement. A single point holding position. Isolated: the state is self-sufficient, not requiring input from others. No boundary needed (⁻b) because there's nothing to push against. Highly stable.
Peaceful
F: gradient N: gradient⁻b D: stable P: sp1D
Peace differs from contentment in that it's relational to an environment — a dissolution of tension between self and surroundings. F:gradient because the representational boundary diffuses. N:gradient⁻b for the same reason. Slightly more spread than sp0D.

F: collapse N: contrastive⁺b D: terminal P: den2D
Desperation is collapse with urgency — the representational field is failing but has not yet ended. Terminal D because it's moving toward a definitive end-state (resolution or exhaustion). Sharp boundary (⁺b) despite the collapse — the self is still trying to maintain a foothold.

Pair 6 — neg · low arousal
Grief
F: loop N: isolated⁻b D: recursive P: med2D
Grief returns repeatedly to the lost object — it is structurally a loop. Isolated: grief is fundamentally a private process even when shared. Recursive D: the particle returns to its prior position, re-experiencing. Denser than sadness because grief carries the mass of the lost representation.
Sadness
F: point N: isolated⁻b D: drifting P: sp1D
Sadness is a low-arousal negative state without a specific loop-object. Point form: the activation is localised and not going anywhere in particular. Drifting D: sadness shifts, lifting and returning without strong recursive pull. Lighter than grief — it doesn't anchor to a specific lost representation.
